打造你的个性“blockquote”

<blockquote> 标签可定义一个块引用，简单的说就是让一个部分突出显示。

wordpress默认模板的blockquote

blockquote {
    margin: 15px 30px 0 10px;
    padding-left: 20px;
    border-left: 5px solid #ddd;
}
blockquote cite {
    margin: 5px 0 0;
    display: block;
}

现在我的主题所用的：

blockquote {
    border: 1px solid #AAAAAA; display:block;
    padding: 5px 5px;
    margin: 1em 20px;
    background-color: #fffff0;
    color: #f2984c;
    font-family:"Courier New", Courier, mono;
    -moz-border-radius: 15px;
}
blockquote:hover {
    background-color:#F7FEDC;
}
code {
    display: block;
    font-family: Courier New;
    font-size: 9pt;
    overflow:auto;
    background: #f0f0f0 url() left top repeat-y;
    border: 1px solid #ccc;
    padding: 20px 5px 20px 20px;
    max-height:none;
    line-height: 1.2em;
    margin: 5px 0 0 15px;
}
code:hover {
    background-color:#F7FEDC;
}

将这段代码插入到模板的style.css文件中就可以了。
