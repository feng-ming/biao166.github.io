利用word保存网页中大量图片（并保持原有顺序）

当我们平时浏览网页逛论坛的时候，经常会遇到不少好图片，想予以收藏。通常我们的做法是在图片上右键——另存为进行保存，或者直接按住左键拖动图片到某个文件夹，这对于保存少量图片的确是很方便，如果遇到了大量的图片呢，会存图存到手抽筋的。

对于多图，比较传统的方法就是把网页直接保存，然后再到保存的文件夹里找自己想要的图，或者用迅雷下载全部链接，文件格式选图片的。但这些方法都还得自己在一堆图片中寻找自己所想要的，图片不按顺序排列且文件命名混乱。

我想我追求的是一种一次性批量保存并且能保持原有顺序的高速快捷方法。我这里讲的方法是用Word2007（word2003也没关系，不过要装一个2003-2007的升级补丁）。

为什么2007可以，而2003不可以呢

MicrosoftOfficeWord2007提供了一种默认文件格式，叫做MicrosoftOfficeWordXML格式（docx的x就是XML）。这种新的文件格式将文件分成一些文档部件，其中每个部件定义了文件的部分内容，当您希望更改文件当中的一些内容的时候，您可以简单的查找您希望更改的文档部件，图片是其中的一种部件，利用这个原理我们来保存图片。也就是我们利用winrar软件读取出这个里边的文件夹，再保存出来，看着有点复杂，但操作起来还是挺快捷的。

软件版本号解析

软件名称后面经常有一些英文和数字，如：QQ 2010 Beta，这些都是软件的版本标志，通过它，我们可以对软件的类型有所了解：

Beta(β)：测试版，为正式版本推出之前发布，可能存在一定的Bug。
Alpha(α)：内部测试版，通常在Beta之前，也存在一定的Bug。
=Retail：零售版。
Free：免费版。
Lite：精简版，也可称迷你版。
Final：正式版，通过Alpha和Beta两个测试版本后推出的，修正了软件中存在的Bug。
Pro：专业版，需用户获得注册码后才可以使用。
Plus：加强版，在功能上有所提升。
Delux：豪华版，和Plus没有太多的区别，也只是多了一些附加功能。
Build：为内部标号，也就是同一版本可以有多个标号，如发V1.0　Build 203等，一般情况下Build后面的数字越大，其版本越新。

让Firefox优化得跟Chrome一样快

firefox一向以启动速度慢著称，但这种现象是可以改变的，下面提供了优化方法让Firefox 优化得比Chrome更快：

1、安装BleachBit或安装插件：(选择下载多语言版，支持Linux和Windows，已包含在Ubuntu 和Debian软件仓库内)，专门用于释放磁盘空间的开源工具，可清除Firefox的缓存和历史，改进启动速度。或者安装插件AFOM Addon,也可以有效地释放缓存。

2、改变渲染：在地址栏输入about:config，改变设置以更快的渲染网页：创建新的整数值content.notify.backoffcount，设定值为5；创建新的值nglayout.initialpaint.delay，设定为0。第一个值是让Firefox不要等到整个网页全部下载后再去渲染，第二个值是改进渲染速度，不要等待网页的布局信息全部下载后再渲染。

3、优化历史：打开about:config，将储存的浏览历史过期天数browser.history_expire_days从默认的180天从改为 7天，browser.history_expire_days_min从默认的90天改为7天（两个值也可以该为14或其它较低的天数），browser.history_expire_sites定义储存的最大网站数量从默认的40000降到更少。

4、加速菜单：打开about:config，创建新整数ui.submenuDelay，值设为0。

Nokia充了很久的电但仍显示电量不足的原因及电池充电的分析

由于平常没养成习惯，默认的手机线充几乎没用，平时多用的是万能充，经常因为充电过久手机显示电量不足，这又容易使我们对电板是否充满不能形成辨别，今天我们特地来讨论下电池充电。

nokia的手机电池有一个特点，那就是如果过充之后，电池电量无法正常显示。这是电池的一种自我保护现象，虽然我不知道什么原理，但是造成这种现象的原因是知道的，那就是电池过充了！！！造成过充的一个主要的原因是，你使用了万能充或者其他的劣质充电器。

我们可以看到电池上通常会标明“充电限制电压：4.2V”，这并不是指充电器的输出的电压，而是指电池的最高限制内压，即电池正负极之间的电压，一 旦超过这个限制电压，那么电池离损坏也就不远了。而损坏的一个表现就是电池鼓包（我管它叫怀孕）。

当你用线充使用手机充电，或者使用 质量好的座充进行充电时，由于手机或座充本身带有完善的电压控制线路，他会检测电池的内压，当达到一定值时就会停止充电，这时，直观的来 说，你会发现线充或者座充的温度降下来了，甚至一点也感觉不出热来。这时，充电完成，即使一直连着电源，也几乎不会对手机造成任何伤害。（当然还是提倡冲 完就拔电源）

word之回车符与换行符探讨

回车符

：分成两个段落了；
组合键:Enter；
别称：硬回车或段落标记；
表示：^p；

换行符：

仅仅把一个长行截断成两行，但还属于同一个段落；
组合键：Shift+Enter；
别称：软回车或手动换行符；
表示：^l；

区别：在于将来当用户改变版面的左右边界重新排版时会自动地删除原有的软回车符号并在新的行末产生一个软回车符号，但无论怎样改变板面，硬回车的地方永远是换行。

如何隐藏：点击“工具”菜单——“选项”——再选择“视图”选项卡，“格式标记”那一栏把“段落标记”前面那个勾取消就可以了。

相互替换和删除：在word进入“查找和替换”——“高级菜单”——“特殊符号”，在查找中输入“^l”（手动换行符），替换中输入“^P”（段落标记）即可全部替换。

重装XP系统时安装SATA驱动的三种方法

背景：SATA 全称是 Serial Advanced Technology Attachment（串行高级技术附件，一种基于行业标准的串行硬件驱动器接口），是由Intel、IBM、Dell、APT、Maxtor和Seagate公司共同提出的硬盘接口规范。

重装 xp 碰到需要安装 SATA 驱动的确是件比较麻烦的事~

1. 首先需要把 BIOS => AHCI HOST CONTROLLER 这一项在安装之前设为 DISABLE（系统安装盘需要集成了 sata 驱动的）;

2. 安装结束后，大家先把官网上的驱动一一安装上去，应该是在主板驱动装好后，在设备管理器中找到IDE ATA/ATAPI控制器这一项，打开后应该会显示几个控制器的菜单，找到一个后缀是2828 的控制器，然后右键选择更新驱动程序，在弹出来的对话框中，选择“从列表或指定位置安装”接下来是要选择“不要搜索，我要自己选择要安装的驱动程序”。（在此之前要去官网上把 SATA DEVICE DRIVER 这个驱动下载解压好）,在弹出的菜单中，下拉列表会看到有一个intel(r)82801HEM/HBM SATA AHCI CONTROLLER，选择他然后点击“从磁盘安装”；

3. 然后就是用到刚才解压好的那个 SATA DEVICE DRIVER 中的一个名称为 iastor 的文件，选择他以后下一步安装，中间会提示“此驱动可能存在兼容风险”大家不用理会它；

4. 安装好以后，系统会提示重新启动计算机，在启动前大家再检查一下刚才的那个 2828 的控制器是否更换为了intel(r)82801HEM/HBM SATA AHCI CONTROLLER，确定无误，就重新启动；

5. 然后在 BIOS 中 ADVANCE 设置中把那个 AHCI HOST CONTROLLER 设置为 ENABLE 吧，这样就不会出现进入 XP 蓝屏，并且使 SATA 硬盘的性能充分的发挥了。
