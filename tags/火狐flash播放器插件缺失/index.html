火狐flash播放器插件缺失

9月 15 2010

火狐提示缺失插件的解决方法（火狐flash播放器不能用）

大多数火狐控们都知道，火狐提示缺失插件的情况比较多，如遇到这种情况我们按照提示选择搜寻所缺失的插件时却找不到，而且多次都无法找到，这时候我们其实可以手动安装插件。

丢失的插件无所谓这几种：

Adobe Flash Player

RealPlayer

Windows Media Player

Shockwave

QuickTime

Java

Adobe Reader
而有以第一个Adobe Flash Player出现的问题最多，这就是所谓火狐flash播发器不能用或者是火狐flash播放器插件缺失。

解决方法：点击进入Firefox的附加组件 直接选择相应插件安装，如出现flash不能播放或者flash的界面提示缺失插件，我们就可以选择Adobe Flash Player的现在下载，进行手动安装。下一步到底（注意中间去掉附带的插件选项），问题就可以解决了。

同样，进入某些音乐网站出现音乐不能播放时我们应该安装的是Windows Media Player插件。
