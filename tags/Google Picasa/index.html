免费图片工具Picasa 十个使用技巧挖掘

Picasa来自Google的免费图片管理工具，Picasa3这个新版本里不断加入了很多有用的功能，能提高用户的图片管理效率。很多人都用不惯这个漂洋过海来的软件，因为可能与我们国人的理解模式不一样，但它却是我用过第二强大的图片管理软件（第一强还没发现）。
我们在用PICASA的时候，经常遇到一些问题，解决了下面的问题后，你应该会用的顺手多了

1、问题：图片哪里去了?

这可能是由3个原因造成的：

(1)可能图片大小小于250*250像素。Picasa默认不显示小于这个尺寸的图片，所以有时你会发现无论如何都找不到这些图片。要让这些图片显示出现只需点击【视图】，勾选“显示小图片”。

(2)可能是Picasa不支持图片的格式，或你没有勾选Picasa显示某种格式的图片。你可以在【工具】-【选项】-【文件类型】里选择显示的格式。

(3)可能是你的图片不在Picasa扫描的范围。你可以在【工具】-【文件夹管理器】里进行设置picasa要扫描的文件夹。

2、批量重命名

选中所有要重命名的图片，按下F2即可。（好像windows里边都一样吧）

3、Picasa不会搜索隐藏文件(夹)

所有包含隐藏属性的文件或文件夹Picasa都不会扫描的。所以，你不用害怕你的绝密照片被Picasa暴露。

Google Picasa相册打不开解决办法！

========= Update 20130421 ========
分享一份云 hosts 文件 名为 SmartHosts。
SmartHosts 是一个托管在谷歌代码上的项目，您可以轻松利用本项目使用到一份稳定的Hosts文件。这份Hosts文件可以帮助您顺利打开一些网站，提高某些国外服务的打开或下载速度。

========= 原文 ========

google的Picasa相册最近由于某些原因打不开。这让很多人包括我感到不便,接下来提供一个方法：

第一步找到 C:\WINDOWS\system32\drivers\etc 文件下hosts文件

第二步，用记事本打开,看见一行 127.0.0.1 localhost

将如下代码插入到后面

203.208.39.104 lh1.ggpht.com


203.208.39.104 lh2.ggpht.com


203.208.39.104 lh3.ggpht.com


203.208.39.104 lh4.ggpht.com


203.208.39.104 lh5.ggpht.com


203.208.39.104 lh6.ggpht.com


203.208.39.104 lh7.ggpht.com


203.208.39.104 lh8.ggpht.com


203.208.39.104 lh9.ggpht.com

关闭文件，保存,就可以重新打开google相册了

VISTA系统会提示无法保存

解决方法：先在附件里以管理员身份运行记事本，然后使用菜单里的“打开”来打开Hosts文件就可以正常保存了。
