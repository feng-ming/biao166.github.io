打造你的个性“blockquote”

<blockquote> 标签可定义一个块引用，简单的说就是让一个部分突出显示。

wordpress默认模板的blockquote

blockquote {
    margin: 15px 30px 0 10px;
    padding-left: 20px;
    border-left: 5px solid #ddd;
}
blockquote cite {
    margin: 5px 0 0;
    display: block;
}
