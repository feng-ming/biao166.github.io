iis，server error

2月 15 2011

server error in '/' application错误

偶然在调试.NET搭建的站点时出现了server error in ‘/’ application错误错误，接连排除了IIS和.net的可能性错误，却还是没找到答案。

突然想到我刚才把站点的根目录文件夹移动了位置，由此可能会导致user用户的权限变化问题，赶紧看了一下，正是此原因，新文件夹已经没有修改的权限，添加上修改的权限后，站点正常了。中间关于排除iis和.net的可能性错误，网上一大把的方法，不在累赘。
