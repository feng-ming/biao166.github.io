在桌面点“刷新”？没用的！

我们都有都有这样的习惯：在Windows桌面右键菜单上狂点“刷新”，总以为能加快系统的速度！！！但真的是那样吗？

先来说说“刷新”的本意。Windows XP帮助文档中的解释为“刷新用当前的数据更新显示信息”。这一功能适用于Windows的很多场合，如在Explorer中更新文件信息的显示，在浏览器中重新载入页面等等。桌面是一个特殊的文件夹，其刷新的本质仍然是更新显示信息。

举例说明“刷新”的正规用途：你正在用Explorer浏览某个文件夹（当然也包括什么也不打开，就盯着桌面这个特殊的文件夹），这时如果某些后台程序（如FTP上传）在这个文件夹中生成了一些新的文件，却没有通过消息循环机制通知Explorer，你就需要手动刷新以看到新文件。再如你正在用FlashGet下载一个文件，打开它所在的文件夹，使用详细信息视图，然后多次刷新试试，你将看到该文件的大小不断增长。在更新显示信息的同时，Explorer也会按用户预设的方式重新排列图标，保持视图整齐有序。

“刷新”也有一些“副业”。由于它通知Explorer进行某些初始化操作，可以使部分与Explorer有关的注册表的项设置的更改立即生效；有时Explorer出现异常，在桌面上刷新一下可能恢复；有时显卡或应用程序出现问题导致桌面上出现“色块”，也可以被刷新去除。

关于装机的那些事

此篇观点完全是从菜鸟立场出发，由来是给人装系统发牢骚来着，虽说现在科技越来越发达，但菜鸟还是有滴，我一边给他装系统，一边还要唠叨。我勒个去！！！

下面来扯扯装机：

系统，笼统说，就是两种：微软正规安装版本和省事的“GHOST”（恢复式）版本。

二者比较起来，前者原汁原味，后者进行了必要的精简和优化。

我倾向于用“GHOST”（恢复式）版本，理由是：1、XP系统需要精简与优化，而这项工作只有专业技术人员才能做到万无一失。2、“GHOST”（恢复式）版本不仅解决了系统优化问题，而且大大简化了系统安装程序，为用户节约了时间和精力。

细分起来，“GHOST”（恢复式）版本又分为“纯净版”、“集成版”、“精简版”、“美化版”等等。

选择“GHOST”（恢复式）版本时，我倾向于选择“集成版”。理由：咱们用排除法——“精简版”一般不要用，因为你很难弄清究竟减掉了一些什么东东；试想一下：当你需要做什么时，系统却没有这个功能，你自己还搞不清楚怎么着解决；烦不烦。用“纯净版”吧，它不安硬件及即插即用设备驱动不说，什么都得从头打闹，小到解压缩软件，大到媒体播放软件，一件又一件，也真够让人烦的。还有，有的纯净版为加快系统各项运行速度，进行了一些特殊优化，比如关闭硬件通道之类，所以不太适合菜鸟。“美化版”呢，更没有必要了，有的系统安装后，主题装了一大堆，有的光占空间就是百十来兆，一项项打开看看，自己喜欢的基本没有，咳，还是个烦。

惠普重装系统小经验总结

以前也给这台电脑（惠普CQ35-217tx）装过系统，这次的重装却花了我n多宝贵的时间，真是恼火……下面大概说下整个过程吧：

第一步：
因为我这没有支持sata的光盘，就只好去深度论坛下了官方深度PE 4.1的，用ultraiso将硬盘映像刻录进U盘，再复制一个Ghost镜像进去，和我电脑上的ultraiso安装文件夹进去，因为我发现pe里边的ultraiso不支持虚拟光驱，这是第一步。

第二步：
第一次用U盘装系统，特意去找了教程，但跟教程中出了点点问题，硬是没装好，后来就只好自己想办：我的想法就是用在PE中用Ghost来装系统 ，首先进入PE后将第一步的映像文件和ultraiso文件夹都复制到电脑的硬盘里，并加载到虚拟光驱，在PE中找到Ghost，按照普通的Ghost来读取镜像并安装，具体可参考惠普等电脑如何使用GHOST做系统备份和恢复 来操作，注意其中需要加载是的.gho文件，在先前加载的虚拟光驱里边有两个.gho文件，选择那个ghost.gho的。至此第二步完成，系统安装中。

重装XP系统时安装SATA驱动的三种方法

背景：SATA 全称是 Serial Advanced Technology Attachment（串行高级技术附件，一种基于行业标准的串行硬件驱动器接口），是由Intel、IBM、Dell、APT、Maxtor和Seagate公司共同提出的硬盘接口规范。

重装 xp 碰到需要安装 SATA 驱动的确是件比较麻烦的事~

1. 首先需要把 BIOS => AHCI HOST CONTROLLER 这一项在安装之前设为 DISABLE（系统安装盘需要集成了 sata 驱动的）;

2. 安装结束后，大家先把官网上的驱动一一安装上去，应该是在主板驱动装好后，在设备管理器中找到IDE ATA/ATAPI控制器这一项，打开后应该会显示几个控制器的菜单，找到一个后缀是2828 的控制器，然后右键选择更新驱动程序，在弹出来的对话框中，选择“从列表或指定位置安装”接下来是要选择“不要搜索，我要自己选择要安装的驱动程序”。（在此之前要去官网上把 SATA DEVICE DRIVER 这个驱动下载解压好）,在弹出的菜单中，下拉列表会看到有一个intel(r)82801HEM/HBM SATA AHCI CONTROLLER，选择他然后点击“从磁盘安装”；

3. 然后就是用到刚才解压好的那个 SATA DEVICE DRIVER 中的一个名称为 iastor 的文件，选择他以后下一步安装，中间会提示“此驱动可能存在兼容风险”大家不用理会它；

4. 安装好以后，系统会提示重新启动计算机，在启动前大家再检查一下刚才的那个 2828 的控制器是否更换为了intel(r)82801HEM/HBM SATA AHCI CONTROLLER，确定无误，就重新启动；

5. 然后在 BIOS 中 ADVANCE 设置中把那个 AHCI HOST CONTROLLER 设置为 ENABLE 吧，这样就不会出现进入 XP 蓝屏，并且使 SATA 硬盘的性能充分的发挥了。
