jQuery版的图片自适应缩放

在幸福收藏夹看到《关于图片按比例自适应缩放》比较感兴趣，最近刚接触jQuery,突然想我是否可以改用jquery改写呢，于是就有了本文，欲看原文请从上边链接前往。

$(document).ready(function(){
resizeImg = function(img, maxWidth, maxHeight){
var w = img.width(),
h = img.height();
// 当图片比预览区域小时不做任何改变
if(w < maxWidth && h < maxHeight) {
return;
}else if( w/h > maxWidth/maxHeight){
img.attr(“width”,maxWidth)
img.attr(“height”,h(maxWidth/w))
}else{
img.attr(“height”,maxHeight)
img.attr(“width”,w(maxHeight/h))
}
}
$(“.cnt img”).each(function(){
resizeImg($(this), 500, 300);
})
})

cnt为外层div，请前往jQuery版的图片自适应缩放看些效果吧，哈哈，

衍生：将博客中图片重设大小，即超过文章内容区域的图片重设为设定大小，代码请看
