Discuz 前端构架——理解（1）

ps：玩discuz快两个月了，按照自己的方式陆续整理出一点可能只有自己看的懂的东西，慢慢的摆出来吧，bs一下。

CSS书写顺序

按照元素模型由外及内，由整体到细节书写，大致分为五组

位置：position,left,right,float

盒模型及属性：display,margin,padding,width,height

边框与背景：border,background

段落与文本：line-height,text-indent,font,color,text-decoration…

其他属性:overflow,cursor,visibility…
常用变量：
SMFONT 小号字体

SMFONTSIZE 小号字体大小 0.83em

NOTICETEXT 提示信息颜色 F26C4F 橙色（有点红）

HIGHLIGHTLINK 高亮链接颜色 369 蓝色

TABLETEXT 普通文本颜色 444

MIDTEXT 中等文本颜色 666

LIGHTTEXT 浅色文字 999

重置

常规的重置，不列出

布局

.cl:after { content: “.”; display: block; height: 0; clear: both; visibility: hidden; } .cl { zoom: 1; }

.z { float: left; } .y { float: right; }

/ Page header / .ph { font-size: 20px; }

/ Main title / .mt { padding: 10px 0; font-size: 16px; }

.pipe { margin: 0 5px; color: #CCC; }

.xs3 { font-size: 16px !important; }

xg和xi

// 提亮级别，两个级别 官方说是两个级别，但这里让人有点误解

.xg1, .xg1 a { color: {LIGHTTEXT} !important; } //浅色 999

.xg1 .xi2 { color: {HIGHLIGHTLINK} !important; } //高亮 369

.xg2 { color: {MIDTEXT}; } //中等文本颜色 666

.xi1, .onerror { color: {NOTICETEXT}; } //橙色 F26C4F

.xi2, .xi2 a, .xi3 a { color: {HIGHLIGHTLINK} ; } //高亮 369

xw

//加粗，两个级别

.xw1 { font-weight: 700; }
.rq{ color:red; } //常用，归于字体设置里边

下边框

//做间隔线用

.bbda { border-bottom: 1px dashed {COMMONBORDER}; } //实线

.bbs { border-bottom: 1px solid {COMMONBORDER} !important; } //虚线

去除边框

.bw0 { border: none !important; }

.bw0_all, .bw0_all th, .bw0_all td { border: none !important; } //适用于表格

去除背景

.bg0_c { background-color: transparent !important; }

.bg0_i { background-image: none !important; }

.bg0_all { background: none !important; }
